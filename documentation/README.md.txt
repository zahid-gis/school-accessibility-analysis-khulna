# 🏫 School Accessibility Analysis in Khulna District, Bangladesh

## 📌 Project Overview

This project analyzes school accessibility in Khulna District, Bangladesh, using QGIS and OpenStreetMap (OSM) data. A 500-meter accessibility buffer was created around each school to identify areas with convenient access to educational facilities and areas with limited accessibility.

The project demonstrates core GIS techniques including data preparation, spatial analysis, geoprocessing, and professional map design.

---

## 🎯 Objectives

- Identify the spatial distribution of schools in Khulna District.
- Analyze school accessibility using a 500-meter buffer.
- Calculate the total accessible area.
- Measure the road network located within accessible zones.
- Identify underserved areas lacking nearby schools.

---

## 🛠 Software

- QGIS 3.x
- OpenStreetMap Data
- Geofabrik Bangladesh Extract

---

## 📂 Data Sources

- OpenStreetMap (OSM)
- Geofabrik Open Data

---

## 🗂 Project Workflow

1. Download OSM Data
2. Extract Khulna District Boundary
3. Extract School Locations
4. Clip Schools to Study Area
5. Create 500 m Buffer
6. Dissolve Buffer
7. Clip Road Network
8. Calculate Road Length
9. Calculate Coverage Area
10. Create Final Map Layout

---

## 📊 Results

| Indicator | Value |
|------------|---------|
| Study Area | Khulna District |
| Number of Schools | 960 |
| Buffer Distance | 500 m |
| School Coverage Area | 529.90 km² |
| District Area | 4648.74 km² |
